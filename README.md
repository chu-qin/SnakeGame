# SnakeGame
基于Qt6.9.2的C++贪吃蛇游戏，面向对象程序设计课程设计实现，在经典玩法基础上拓展多食物效果、等级联动、BGM切换等功能，采用Qt Widgets架构与模块化设计，实现游戏逻辑与界面展示解耦。

## 开发环境
- 系统：Windows 11
- 工具：Qt Creator
- 框架：Qt 6.9.2（Qt Widgets）
- 语言：C++
- 编译器：MinGW 64-bit

## 核心功能
1. 基础操作：WASD/方向键控移动，空格键临时加速，支持开始/暂停/重新开始游戏
2. 死亡判定：检测蛇头撞墙、撞自身，结束后自动停BGM和计时器
3. 多维度统计：实时显示分数、游戏时间、等级、蛇身长度及各类食物数量
4. 多食物系统：金色（+10分+变长）、绿色（临时加速）、红色（临时减速），效果伴蛇身颜色变化
5. 等级机制：每30分升1级，等级关联基础速度、食物生成数量与概率
6. 音效联动：按分数区间自动切换背景音乐，循环播放
7. 辅助功能：独立帮助弹窗展示游戏规则，模块化UI分游戏区与信息区

## 核心类设计
- **GameWidget**：核心游戏逻辑，实现蛇、食物的运动/生成/碰撞，处理游戏状态、绘图渲染、BGM管理
- **MainWidget**：界面布局管理，实现游戏区与信息区分栏，展示统计数据，绑定功能按钮信号
- **HelpDialog**：游戏帮助弹窗，展示操作、规则、食物效果、等级说明等信息

## 关键规则
1. **等级与参数**：初始速度400ms/步，每级减40ms（7级+为120ms/步）；等级越高食物生成数量越多（最多5个）
2. **食物概率**：金色60%-80%（等级越高概率越低）、绿色15%、红色5%-20%（等级越高概率越高）
3. **BGM切换**：0-59分播放bgm1.mp3，60-119分播放bgm2.mp3，120+分播放bgm3.mp3
4. **操作目标**：吃食物累计分数升级，避免撞墙/撞自身，无需重启程序即可重置游戏
